### To start proxy server

```
REPLICATE_API_TOKEN=[API_TOKEN] PORT=80 python server.py
```

### To run example

```
# edit ENV
# OPENAI_API_BASE=http://localhost/v1 -> to proxy server
# REPLICATE_MODEL_PATH to replicate model path
python example.py
```
